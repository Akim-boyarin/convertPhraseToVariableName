# convertPhraseToVariableName
Данное приложение преобразует словосочетание в имя переменной в формате lowerCamelCase, ориентированном в частности на написание кода на JavaScript. Логика реализована с помощью Vue.js, вёрстка HTML/CSS, методология БЭМ. Подключена дополнительная библиотека copy-js (https://www.npmjs.com/package/copy-js).
